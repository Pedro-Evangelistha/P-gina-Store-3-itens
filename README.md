PÁGINA DEVCLUB STORE

Projeto de uma página simples de loja com **3 itens**, desenvolvido com base em um vídeo-curso do DevClub do Rodolfo Mori. A proposta é praticar e consolidar conhecimentos em **HTML5**, **CSS3** e **JavaScript**


No projeto, você encontrará:

- Uma página estática com layout responsivo simples.
- Três produtos exibidos (imagens + nome + preço).
- Funcionalidade básica com JavaScript (por exemplo: ao clicar em “adicionar ao carrinho”, ou cálculo, ou interação).
- Comentários no código para facilitar o entendimento do que cada parte faz.
- Arquivos principais:
  - `index.html` — estrutura da página
  - `styles.css` — estilos visuais
  - `scripts.js` — lógica / interatividade
  - imagens dos produtos (ex: `air-pods.png`, `apple-watch.png`, `vision-pro.png`)
 
  ***

TECNOLOGIAS USADAS

- **HTML5** — marcação semântica e estrutura da página  
- **CSS3** — posicionamento, flexbox, responsividade, design visual  
- **JavaScript** — manipulação do DOM, eventos, lógica de interação

DESAFIOS QUE ENFRENTEI

`Como estruturar o HTML de forma clara e semântica.`
Boas práticas de CSS (nomes, modularização, responsividade).
`Manipulação do DOM com JavaScript puro (sem frameworks) para adicionar funcionalidades básicas.`
A importância de comentar o código para facilitar leitura e futura manutenção.
`Desafios: compatibilidade entre navegadores, ajustar responsividade para diferentes tamanhos de tela, melhorar usabilidade.`
